Difference bw preload and prefetch in terms of parsing and execetation
The main difference between preload and prefetch is that preload tells the browser to download and parse a resource as soon as possible, while prefetch tells the browser to download a resource but does not require it to parse it. Here is a table that summarizes the difference:

table content is may not true

Characteristic Preload Prefetch
Downloads the resource Yes Yes
Parses the resource Yes No
Executes the resource No No

When you preload a resource, the browser will download and parse it as soon as possible. This means that the resource will be available to the page as soon as it has loaded. This can be useful for improving the performance of your web application if the resource is essential for the initial loading of the page.

When you prefetch a resource, the browser will download it but will not parse it. This means that the resource will not be available to the page until it has been parsed. This can be useful for improving the performance of your web application if the resource is not essential for the initial loading of the page but is likely to be used later.
